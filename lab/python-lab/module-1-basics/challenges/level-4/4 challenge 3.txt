# Module 1 - Basics
## Level 4 - Challenge 3

Problem
Take a number from the user and print the cube of the number.

Instructions
1. Ask the user to enter a number.
2. Convert it into integer.
3. Multiply the number three times.
4. Print the result.

Example Input
3

Example Output
27

Hint
Cube = number * number * number

Broken Code
num = int(input("Enter number: "))
print(num * num)

Correct Code
num = int(input("Enter number: "))
print(num * num * num)
